# Arkanoid Pro 2025

**Arkanoid Pro 2025** is a modern take on the classic Breakout genre, developed as a group project for the Object-Oriented Programming (OOP) course. The game challenges players to destroy walls of bricks using a bouncing ball and a movable paddle, featuring enhanced mechanics and a cooperative mode.

## Key Features

### Gameplay Mechanics

* **Classic & Smooth:** High-precision collision detection between the ball, paddle, and bricks for a satisfying gaming experience.
* **Level Progression:** 5 increasingly difficult levels with unique brick layouts and challenges.
* **Co-op Mode:** Supports local 2-player gameplay, allowing friends to work together to clear stages.

### Power-Ups System

Enhance your paddle with various drops:

* **Laser Gun:** Shoot down bricks directly from your paddle.
* **Enlarge:** Increase paddle width for easier ball catching.
* **Health Recovery:** Restore lost lives to stay in the game longer.
* **Speed Buffs:** Dynamic ball speed adjustments.

### Customization & Persistence

* **Custom Entities:** Adjustable ball physics and diverse brick types (different durability/points).
* **Save/Load System:** Resume your progress anytime.
* **Leaderboard:** Track high scores and compete for the top spot.

## Tech Stack

* **Language:** Java
* **Library:** Java Swing / AWT (for Graphics and UI)
* **Tools:** IntelliJ IDEA, Git
* **Design Pattern:** Applied OOP principles (Inheritance, Polymorphism, Encapsulation, and Abstraction).
